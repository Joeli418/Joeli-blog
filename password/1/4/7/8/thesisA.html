Application of Generalized Reed–Muller Expression for Development of Non-Binary Circuits

This paper talks about programmable logic arrays in non-binary logic. Where a unit block is made from a memory section and a logic section. The memory section uses Ferroelectric elements and the logic section uses generalized Reed-Muller expression.

High-Efficient Circuits for Ternary Addition

This paper puts a little more focus on speed and power consumption over number of transistors It mentions the use of CNTFET technology and it's performance over MOSFET when it comes to ternary logic design. It also proposed a TFA (ternary full adder) schematic and compares it with two prveiously proposed TFA to compare with, showing it has better performance.

Design and Evaluation of CNFET-Based Quaternary Circuits

Quaternary circuit, again revolved around CNTFET. talks about some design. such as quad buffer, quad max, quad min, quad inverter, decoder and multiplexer. talks about how these circuits can be used to construct adders and multipliers.

Design and application of precise analog computational circuits

a current approach to implement a squarer and a square rooter with low distortion as it's main explores ways to minimize distortion.
